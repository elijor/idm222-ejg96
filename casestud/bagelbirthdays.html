Overview

Bagel Birthdays was a javascript project I designed and developed in Winter 2023 to further my knowledge of interactive websites, modern browser-based interfaces, and scripting frameworks.

This project introduced me to various skills in developemnt, especially in javascript. I also explored various animations and transitions through a developer's persepctive. Finally, I utilized Figma, Adobe Photoshop, and Adobe Illustrator to create the graphics and layout for this application.

Prompt

Create a web-based application framed around the 12 western astrological zodiac signs. The 12 symbols do not need to be actual zodiac icons, but any 12 objects, characters, or concepts within a theme. Users will be able to input their birthday and receive a corresponding symbol and information back.

Final Product Must:

  1. - Allow users to input birthday
  2. - Have 12 distinct icons with corresponding information
  3. - Have an optional help button to explain to users how to interact with the app
  4. - Be fitted for desktop usage
  5. - Each icon must emit a unique noise once selected

Javascript and Development

Using Visual Studio Code (VS Code), I wrote the html, css, and javascript code. I coded each of the 12 bagels to correspond with a zodiac sign, based on the user’s inputted birth month and day. Each bagel has a unique name, image, description, and sound effect.

I also created some aesthetic transition effects as well. Upon selecting one bagel, the others will gray out. While the others are gray, hovering over a gray bagel with fade it to color and slightly enlarge the bagel.

Finally, I added 12 unique sounds to play when a bagel is selected through click or by entering birthday. I ensured that one it played a sound, the one before would cut off so the sound effects wouldn’t overlap.
